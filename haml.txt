copper:
  - HTML tags (%p)
  - nesting tags (indentation)
  - tags with class(es) and/or ids defined (.#)
  - embed ruby in tag (with output/no output: '=' vs '-')
  - ruby interpolation (#{})
  - tag attributes
  - doctype (!!!)
  - ruby blocks (each loop)
  - comment line (/ and -#), difference between them
silver:
  - boolean attributes (arg: true)
  - HTML5 data attributes (data: {a: 2})
  - multiline HTML args, ruby code
  - filters (:javascript)
  - helpers (surround, precede, succeed)
  - empty tags (br/)
  - comment block (/ and -#)
  - escape line (\)
gold:
  - mulitple class/id parts (with ruby array syntax)
  - Active Model object reference ([])
  - IE conditional comments (/[])
  - (un)escape HTML (!=, &=)
  - whitespace removals (<, >)
  - multiline text (|)
